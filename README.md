
# 水浒·星引缘 (Shuihu Heart: Moonlight at Liangshan)

一款以《水浒传》为背景的恋爱 AVG 文字冒险游戏，集成 Gemini AI 实现动态角色互动。

## 🚀 快速开始

1. **上传到 GitHub**: 将此仓库上传至你的 GitHub。
2. **部署到 Vercel**: 
   - 关联此 GitHub 仓库。
   - 在 **Environment Variables** 中添加 `API_KEY`（填入你的 Gemini API Key）。
   - 点击 **Deploy**。

## 🛠️ 技术栈
- **React 19**: UI 框架。
- **Vite**: 构建工具。
- **Tailwind CSS**: 样式处理。
- **Gemini API**: AI 对话引擎。

## 📖 玩法说明
- **剧情推进**: 点击对话框体验梁山初遇。
- **传信系统**: 在“名册”或悬浮窗找英雄聊天，AI 会根据性格实时回复。
- **卜卦系统**: 每日可找公孙胜求一卦，获得好感加成或特殊效果。
